# 🎵 Piano Pals

A fun, interactive web app that teaches kids (ages 3–7) how to play the piano. Just prop your iPad near the keyboard, tap the microphone button, and start playing!

## Features

- **Microphone-based pitch detection** — no cables or adapters needed. The app listens through your device's microphone and detects which notes are being played on a real piano/keyboard.
- **Two kid profiles** — "Little Star" (toddler mode, forgiving) and "Rock Star" (big kid mode, scoring & streaks).
- **Free Play mode** — play any keys and see colorful note names, rainbow particles, and fun emoji reactions.
- **Lesson mode** — learn 5 classic songs with a scrolling note guide:
  - Twinkle Twinkle Little Star
  - Mary Had a Little Lamb
  - Hot Cross Buns
  - Ode to Joy
  - Jingle Bells (Chorus)
- **Parent Settings** — toggle difficulty, unlock/lock songs, switch between microphone and USB/MIDI input, adjust volume, and reset progress.
- **Confetti & celebrations** — star ratings and confetti animations when kids complete a song.
- **Fully responsive** — works great on iPads, tablets, and phones.

## How to Use

1. Open `index.html` in any modern browser (Safari on iPad works great).
2. Tap the big green **"Tap to Turn On Microphone"** button.
3. Place your device near your piano keyboard.
4. Pick a profile and start playing!

## Tech Stack

- Pure HTML, CSS, and JavaScript — no build step, no dependencies.
- [Nunito](https://fonts.google.com/specimen/Nunito) font from Google Fonts.
- Web Audio API for pitch detection and sound synthesis.
- Web MIDI API (optional, for USB-connected keyboards).

## iPad Setup

1. Open Safari on the iPad.
2. Navigate to the hosted URL (or open the `index.html` file).
3. Tap **Share → Add to Home Screen** to make it feel like a real app.
4. Prop the iPad on or near your keyboard and have fun!
